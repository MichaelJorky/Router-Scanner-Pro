# Router-Scanner-Pro
Router Scanner Pro adalah aplikasi pemindaian router berperforma tinggi yang dirancang untuk melakukan scanning terhadap banyak alamat IP secara paralel menggunakan multi-threading. Aplikasi ini membantu melakukan identifikasi perangkat jaringan, memeriksa port terbuka, mendeteksi otentikasi, serta mengambil informasi jaringan seperti BSSID dll.
